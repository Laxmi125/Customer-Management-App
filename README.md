# Customer management application for Sunbasedata  
The Customer Management Application is a CRUD app built with Java and Spring Boot that allows user to manage customer records. It provides features for creating, updating, searching, viewing, and deleting customer details.  
Assignment given by: [Sunbasedata](https://www.sunbasedata.com/)
## Features
- Create a new customer by providing the customer's name, address, email and mobile number.
- Find a customer by id.
- View a list of all customers.
- Update customer information.
- Delete a customer from the records.
## Tools & Technologies
- Spring Boot 3.2.2
- Java 17
- JPA
- MySQL Database
- Maven
- Postman
